# YS-QuotePress

WordPress  (CPT + PDF + signature) Custom plugin for creating and signing quotes:
- CPT `quote`
- PDF generation (Dompdf)
- Hebrew/RTL fonts (Heebo)
- Signature canvas (SignaturePad)
- Admin table + delete rows

## Dev
- PHP 8.3
- Composer for Dompdf
- Place fonts under `assets/fonts/Heebo/`
